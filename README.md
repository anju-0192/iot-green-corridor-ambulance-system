# IoT Enabled Green Corridor System for Ambulances

## Overview
This project proposes an IoT-based smart traffic management system that automatically creates a green corridor for ambulances and emergency vehicles.

Traffic signals dynamically turn green along the ambulance route to reduce travel time and save lives.

## Features
- Automatic green corridor creation
- Real-time GPS tracking
- LoRa long-range communication
- Smart traffic signal control
- Emergency priority routing

## Hardware Components
- ESP32
- LoRa SX1278
- GPS Module NEO-6M
- IR Sensors
- Relay Module
- Traffic Light System

## Software Components
- Arduino IDE
- Google Maps API
- IoT cloud platform
- Mobile application
